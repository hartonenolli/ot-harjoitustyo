# Viikko

**Maanantai: **
   Käy kaupassa, 
   Vie lapset harrastuksiin

**Tiistai: **
   **Osta lahja _Kakelle_ kaupasta**

**Keskiviikko: **
   ***Kaken synttärit***, 
   Vie koira hammaslääkäriin

**Torstai: **
   Aja nurmikko

**Perjantai: **
   *Lähtö Tallinnaan*

**Lauantai: **
   Tallinnassa

**Sunnuntai: **
   *Paluu Tallinnasta*
